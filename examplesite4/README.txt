# Portfolio Theme

## Overview
This is a simple portfolio theme designed to showcase projects and personal information.

## Structure
- `parts/`: Contains reusable HTML components like header and footer.
- `patterns/`: Contains PHP functions for reusable patterns (e.g., project cards).
- `styles/`: Contains style configurations for the theme.
- `templates/`: Contains template files for different pages (e.g., index, archive, 404).

## Usage
1. Place this folder in your WordPress themes directory.
2. Activate the theme through the WordPress admin panel.
3. Customize the theme by modifying the HTML, CSS, and PHP files.

## Files
- `index.html`: Main homepage template.
- `archive.html`: Archive template for listing projects or blog posts.
- `singular.html`: Template for individual posts/pages.
- `404.html`: Error 404 page template.
- `functions.php`: PHP functions for the theme.
- `style.css`: Main stylesheet.
- `theme.json`: Theme configuration file for block editor styles.
